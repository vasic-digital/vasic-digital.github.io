诱人之处

你的笔记、任务、草稿——一切尽在一处，以简洁、面向未来且真正属于你的格式呈现。Yole 是一款多功能、轻量级的文本编辑器、笔记与待办应用，支持 Markdown、todo.txt、纯文本、数学符号等多种格式，绝不会将你的文字困在任何封闭的专有系统中。

为何引人入胜

Yole 立足于”简单”与”强大”的交汇点——这正是最难驾驭的领域。它是一款毫无花哨的编辑器，启动迅速，不干扰你的思路，却能流畅处理开发者和高级笔记用户所重视的结构化格式：Markdown 用于文档撰写，todo.txt 用于任务管理，数学符号用于技术写作，等等。Yole 由 Kotlin 构建，基于共享代码库覆盖 Android、桌面端、iOS 及 Web 平台，是一款真正跨平台的写作工具——现已在 Android 平台正式发布，通过 F-Droid 和 GitHub 分发。

棘手的难题

纯文本工具入门容易，完善却极为艰难。要良好支持多种文本格式，意味着必须进行实质性的解析与渲染工作——Markdown、todo.txt 和数学符号各有其规则与边界情况——同时还要保持应用轻量且响应迅速。从单一 Kotlin 代码库出发实现多平台覆盖，则需在 Android、桌面端、iOS 及 Web 上提供原生般的体验，而不致分化为四个各行其是的应用。而整个设计的核心原则在于克制：在不牺牲速度与简洁性的前提下，逐步增加功能——这正是让文本编辑器每日使用都令人愉悦的关键。

颠覆性的意义

Yole 为用户提供了一个统一、便携的写作空间，所有内容均以开放格式存储，超越任何应用的生命周期。基于纯文本的设计，确保你的内容始终可读且完全属于你——无锁定，无迁移焦虑。其多格式支持的流畅性，意味着一款工具即可同时充当笔记应用、任务管理器和技术草稿本。作为开源软件，且在 F-Droid 上实现真正分发，Yole 为用户提供了一个注重隐私、实用的替代方案，摆脱那些试图掌控你数据的云端笔记服务。

如何攻克最难的部分

Yole 采用 Kotlin 与基于 Compose 的多平台方案构建，单一代码库即可覆盖 Android、桌面端、iOS 及 Web 平台——在共享逻辑与界面的同时，仍力求在各平台呈现原生体验。多格式支持通过为每种文本格式定制解析与渲染实现，确保 Markdown、todo.txt、纯文本及数学符号各自行为准确，而非仅仅近似模拟。项目以渐进且务实的方式发布——Android 版已正式上线，桌面端处于测试阶段，iOS 与 Web 版正在积极开发——这既保证了已发布版本的精良体验，又持续拓展平台覆盖范围。始终贯穿其中的设计原则是：优先保持编辑器的简洁与速度，在不损害核心写作体验的前提下，逐步叠加格式与功能。
